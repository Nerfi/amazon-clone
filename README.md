Clone of the popular e-commerce website Amazon with the main features of the real app.

## Technologies
  - React
  - Firebase
  - React Context
  - Express
  - Stripe API
  - Bootstrap
  - Firebase storage


### Installation

To run this project, install it locally using npm:

```sh
$ git clone git@github.com:Nerfi/amazon-clone.git
$ cd amazon-clone
$ npm install
$ npm start
```
